Pixel Wheels

Welcome to Pixel Wheels! Pixel Wheels is a retro top-down race game, which you can play alone or with a friend.

It works on Linux, Intel-based macOS, Windows and Android.

Controls

The user interface and the vehicle can be controlled using either gamepads or keyboard.

Vehicles accelerate automatically.

Keyboard

To navigate the menus and to control the first player vehicle, use the arrow keys and the Space key to select menus or activate bonuses.

In 2-player mode, the keys for the player on the right-side of the screen are:

  • X: Left
  • V: Right
  • D: Up
  • C: Down / brake
  • Left control: activate bonuses

Gamepads

You can configure mapping of the gamepad buttons in the configuration menu.
