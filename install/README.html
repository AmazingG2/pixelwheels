Tiny Wheels

Welcome to Tiny Wheels! Tiny Wheels is a retro top-down race game, which you can play alone or with a friend.

It works on Linux, macOS, Windows and Android.

Starting the game

Make sure you have a Java Runtime Environment (JRE) installed. If you don't, you can get one from here.

You should then be able to start the game by double-clicking on tinywheels.jar. If you prefer the command line, run java -jar tinywheels.jar.
